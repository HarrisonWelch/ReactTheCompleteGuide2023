# Revisiting Operators

```js
// ARITHMETIC
console.log(10 + 5);
console.log(10 - 5);
console.log(10 * 5);
console.log(10 / 5);
console.log(10 % 5);
console.log("hello " + "world");

// RELATIONAL
console.log(10 === 10);
console.log(10 === "10"); // Type compare would make false
console.log(10 == "10"); // now its true

console.log(10 >= 5);
console.log(10 <= 5);
console.log(10 > 5);
console.log(10 < 5);

// Usually used in 'if' statements
if (10 === 10) {
  console.log("works");
}
```

5

2
